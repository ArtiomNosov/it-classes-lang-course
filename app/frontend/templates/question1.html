Как сложить две переменные?

Ответил: pele68


Для сложения двух переменных в C++ можно использовать оператор `+`:

            int a = 5;
            int b = 10;
            int sum = a + b;
            std::cout << "Сумма: " << sum;

Пример работает для целых чисел. Если используете другие типы данных, убедитесь, что их можно складывать.
